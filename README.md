ht-java-tomcat
==============

Sample project that use tomcat(webapp-runner) for Heroku training

How to deploy
--------------

### Step1  

Clone app from Github  

```
$ git clone git@github.com:flect/ht-java-tomcat.git
$ cd ht-java-tomcat/
```

### Step2  

Create Heroku application

```
$ heroku create
```

### Step3  

Deploy application
```
$ git push heroku master
```

### Step 4

Visit application

```
$ heroku open
```
